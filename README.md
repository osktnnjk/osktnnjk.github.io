# ワイのゼニはどこ行った for 大阪

wheredoesmymoney.orgの大阪バージョンです．

http://osktnnjk.github.io/ で稼働しているサイトのソースコードです。

Where Does My Money Go? β版（2012年7月1日稼働）は、大阪市民が大阪市に納めている市税を対象として構築したものです。
自分の年間総収入をスライドで設定し、単身世帯か扶養一人世帯かを選択すると、給与所得者であるという前提で、大阪市に納めている市税年総額と10分野毎に一日当たり支払っている市税額が表示されます。
所得控除額は、単身世帯の場合33万円、扶養一人世帯の場合66万円、市税負担額は課税対象額の6％と設定しています。
また、税金の配分比率は、＊＊＊＊の予算配分比率( 年 月 日現在)を使っています。

現在このβ版は、大阪市税だけを対象としたものですが、今後、利用者の皆さんや日本政府や全国の自治体の皆さんのご意見、ご要望を踏まえて、特別会計や公営企業会計も含めたり、市債発行額・償還額を含めたり、都道府県税や国税を含めたりなど、ニーズに応じて、どんどん成長させて行きたいと考えております。是非、皆様からの忌憚のないご意見、ご要望、ご指摘をお願いいたします。
